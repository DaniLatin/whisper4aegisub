How to use

Installation

  1. Download and install Aegisub (if you haven't already) from here:
    https://aegisub.org/
  2. Download the latest version of whisper4aegisub
  3. Extract it
  4. Copy the contents into Aegisub/automation/autoload/ folder
    Usually on Windows:
    C:\Program Files\Aegisub\automation\autoload
  5. Restart Aegisub

Usage

  1. In Aegisub open a video Video ‚Üí Open Video or audio Audio ‚Üí Open Audio
  2. Then in Aegisub open: Automation ‚Üí AI Transcription
  3. Select a language
  4. Select a model (large-v3 is recommended for best accuracy)
  5. Set other settings by your preference
  6. Press Start to start transcribing
  7. A cmd window will pop up, showing the process
  8. Once the job is done the subtitles will be automatically loaded into Aegisub, ready for further editing

Activation

  1. In Aegisub open: Automation ‚Üí AI Transcription
  2. Enter your license key into the license input
  3. Click Activate license
  4. Your license key will be checked
  5. Congratulations! You're now ready to go in full pro mode!

Deactivation

You can use your license key for 3 machines. But if you ever want to transfer your license to another machine, we've got you covered.

  1. In Aegisub open: Automation ‚Üí AI Transcription
  2. Click Remove license
  3. Your license key will be removed and you can now use it on another machine
